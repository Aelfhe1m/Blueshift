Blueshift: Kerbal FTL

---INSTALLATION---

Simply copy all the files into your GameData folder. When done, it should look like:

GameData
	WildBlueIndustries
		Blueshift

New Parts

Mk2 Warp Core
Mk2 Gravimetric Displacement Generator
Mk2 Mini Fusion Reactor
Mk2 FTL Propellant Tank
Mk2 Warp Ring*
Mk2 Warp Coil*
S-1 Warp Coil*
S-1 FTL Endcap Tank
* Has part variants

Changes

Due to their versatility, warp coils now cost more to obtain the same performance as an equivalent Warp Ring.
